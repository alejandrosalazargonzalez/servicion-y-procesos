# Examen de procesos linux e hilos

## Carrea entre Rayo McQueen y Mate a través de hilos

<img src=https://codigoespagueti.com/wp-content/uploads/2022/08/cars-on-the-road.jpg width=300>

Deberás crear una clase CarRace que represente a cada coche en la carrera. La carrera tendrá una meta (un número entero que indica la distancia total a recorrer). Cada coche avanzará en pasos aleatorios, y se debe garantizar que solo un coche se mueva a la vez mediante. Los parámetros que se que tiene el contructor es nombre y distancia.

- name: nombre del coche.
- distance: distancia recorrida por el coche.
- goal: la meta a alcanzar (distancia total).

> Funciona (2 puntos)/Funciona + documentado (2,5 puntos)/Funciona + documentado + optimo (3 puntos.
